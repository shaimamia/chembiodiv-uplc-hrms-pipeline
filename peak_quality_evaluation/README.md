# Peak Quality Evaluation

## Description

This module evaluates the quality of detected peaks from the marker detection step.

---

## Objectives

* Assess peak reliability
* Remove low-quality or noisy peaks
* Validate known compounds (e.g., sulfamethoxazole)

---

## Metrics

* Signal-to-noise ratio
* Peak shape quality
* Consistency across replicates

---

## Inputs

* Detected markers

---

## Outputs

* Filtered high-quality peak list
* Quality scores
