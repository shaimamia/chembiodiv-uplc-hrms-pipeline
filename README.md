
# ChemBiodiv UPLC-HRMS Pipeline

## Overview

This repository contains the data processing pipeline developed during my Master 2 internship at LEESU (ChemBiodiv project).

The goal is to process and analyze UPLC-HRMS data for non-target screening of environmental micropollutants.

---

## Pipeline Steps

1. Data Preprocessing
2. Marker Detection
3. Peak Quality Evaluation
4. Peak Alignment
